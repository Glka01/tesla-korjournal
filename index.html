Python-skript för att hämta token

Kör detta i terminal om du inte har en app:

pip install tesla-fleet-api

python3 - <<'EOF'
from tesla_fleet_api.teslafleetapi import TeslaFleetApi
import asyncio

async def main():
    api = TeslaFleetApi(
        client_id="YOUR_CLIENT_ID",
        client_secret="YOUR_CLIENT_SECRET",
    )
    # Följ OAuth-flödet i konsolen
    print(await api.vehicles())

asyncio.run(main())
EOF

CORS-problem?

Om du öppnar appen som en lokal fil (file://) kan webbläsaren blockera API-anrop p.g.a. CORS-restriktioner. Lösningar:

1. Servera filen lokalt: python3 -m http.server 8080 och öppna http://localhost:8080

2. Använd Chrome med CORS inaktiverat (ej rekommenderat för daglig användning)

3. Importera via JSON-fil (se nedan)
